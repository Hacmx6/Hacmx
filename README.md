Hacmx 🔍

Hacmx é uma ferramenta de OSINT e análise de segurança desenvolvida em Python para auditoria de domínios.

⚡ Funcionalidades

- 🔎 Enumeração DNS
- 🌍 GeoIP lookup
- 🔐 Verificação SSL
- 🚪 Scanner de portas
- 📡 Monitor de conectividade
- 📊 Score de segurança
- 📄 Geração de relatório

🚀 Instalação

git clone https://github.com/Hacmx6/Hacmx.git
cd Hacmx
pip install -r requirements.txt

🧪 Uso

hacmx -d example.com --mode full

📌 Exemplo de saída

- IP
- Subdomínios
- Portas abertas
- Informações SSL
- Localização
- Score de segurança
